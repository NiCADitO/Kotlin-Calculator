## Partnership Project

Calculator built with Kotlin and Swing.

Utilizes **FlatLaf** for a flat dark theme and **MigLayout** for precise grid-based positioning. Application logic handles asynchronous tasks using Kotlin **Coroutines** to ensure the UI remains responsive.

## Repository overview

```text
partnershipproject/
├── src/
│   ├── main/kotlin/.../      # Main application logic (Main.kt, PanelView.kt)
│   └── test/kotlin/.../      # Unit tests (CalculatorTest.kt)
├── build.gradle.kts          # Dependencies (FlatLaf, MigLayout, Coroutines)
├── settings.gradle.kts       # Project settings
├── gradlew                   # Startup script (Linux/Mac)
└── gradlew.bat               # Startup script (Windows)
```
